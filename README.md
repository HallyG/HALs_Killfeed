# HALs Killfeed
Adds a configurable kill ticker to ArmA which tracks deaths and displays them on a client's screen.
- ctrlCreates an **RscControlsGroupNoScrollbars** control on **display 46** with the **idc 3200**.

## Dependencies
- No Addons required.
